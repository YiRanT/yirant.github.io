# 魅力 CHA
## 各职业隐藏魅力

`骑士`  隐藏魅力 +0

`王族` `法师` `黑暗妖精` 隐藏魅力 +6

`妖精` 隐藏魅力 +12

## 魅力控制数量
- 魅力控制包括：宠物、召唤、迷魅
- 普通召唤与宠物控制数量 ( `魅力`+ `隐藏魅力` ) 除以 `6`
- 特殊宠物 `老虎` `高丽幼犬` ( `魅力`+ `隐藏魅力` ) 除以 `8`
- 妖精召唤 `属性精灵` `强力属性精灵` 控制数量 `限定1只`
